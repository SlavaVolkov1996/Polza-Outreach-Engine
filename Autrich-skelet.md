## Описание конструкции: 
- Бессерверное AWS (Lambda Manager/Исполнитель + SQS/DynamoDB/S3) + мультивендор SMTP (SES/SendGrid/Mailgun). 
- 1200 адресов изолированы по клиентам/поддоменам, пулы по 200/аккаунт с прогревом и IP-ротацией.

## Сервисы и подходы: 
- Lambda (оркестрация/отправка), 
- SQS (очереди), DynamoDB (метрики), 
- S3 (логи), Route53 (DNS/SPF/DKIM), 
- Warmup Inbox (20% пула).

## Ротация и мониторинг: 
- Менеджер выбирает адрес (свежий>24 часа, <200 писем в день, доставляемость >90%) из DynamoDB → Исполнитель отправляет → обновить метрику. 
- Алерты CloudWatch/Postmaster (отказов >3% → пауза), Slack, дашборды QuickSight.

## Распределение нагрузки: 
- Провайдеры (40% SES/30% SendGrid/30% Mailgun), 
- циклический перебор по пулам (максимум 240 тыс. писем в день), 
- автоматическое масштабирование Lambda, 
- регулирование SQS (<500/мин).

## Риски и закрытие: 
- Репутация: Разминка 14–30 дней + автоизоляция. 
- Блок: Мультивендор + 20% резерв. 
- Ограничения: счетчики в реальном времени + оповещения об очереди. 
- Время простоя: несколько AZ + неактивные SQS.

## Оценка стоимости (мес, 1 миллион писем): 
- AWS 20–30 долларов, 
- SMTP+разминка 1000–1300 долларов, 
- DNS 20 долларов. 
### Итого: 1050–1350 долларов.
